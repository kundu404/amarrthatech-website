# 🌐 Amarrthatech Website

A responsive multi-page company website built using HTML, CSS, Bootstrap, and PHP (with working contact form).

## 📁 Pages

- `index.html` – Home
- `about.html` – About Us
- `services.html` – Services
- `contact.php` – Contact form (uses `send_mail.php`)
